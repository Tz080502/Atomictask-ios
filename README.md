# TaskFocus iOS App

A minimal iOS app designed for users with ADHD to focus on one task at a time.

## Features

- **One Task at a Time**: Display a single task card to reduce overwhelm
- **3 Swipes Per Day**: Limit task completion to 3 per day
- **Smart Sorting**: Choose between "Quick Tasks First" or "Hard Tasks First"
- **Supabase Integration**: Syncs with your existing task management SaaS
- **Clean UI**: Designed with ADHD-friendly colors and minimal distractions

## Setup

1. Open the project in Xcode
2. The Supabase configuration is already set up
3. Build and run on iOS 17+ device or simulator

## Color Scheme

- Background (Cream): #F9F3E9
- Card Background (White-ish): #FFFCF9
- Accent Red: #B35C5C
- Accent Blue: #608BC1
- Text (Dark Charcoal): #3D3D3D
- Border/Stroke: #E8E2D8

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+
